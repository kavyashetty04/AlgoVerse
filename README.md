# AlgoVerse – USDT Transaction Notification System

AlgoVerse is a Node.js-based application designed to streamline USDT buy and sell request management through an automated notification workflow. The system allows users to submit transaction requests, which are then processed and forwarded to the administrator through detailed email notifications.

## Key Features

* Buy and Sell USDT request submission
* Automated email notifications for every transaction request
* Real-time delivery of transaction details to the administrator
* Structured request handling and validation
* User-friendly transaction workflow
* Backend automation using Node.js

## Objective

The primary objective of AlgoVerse is to simulate a real-world cryptocurrency transaction notification system by reducing manual monitoring efforts and ensuring that transaction requests are instantly communicated to the concerned party.

## Technologies Used

* Node.js
* JavaScript
* Email Services (SMTP/Nodemailer)
* HTML/CSS (Frontend Interface)

## Learning Outcomes

Through this project, I gained practical experience in backend development, request handling, email automation, asynchronous operations, and designing workflow-based applications.
